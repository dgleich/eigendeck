---

The walk profile at m is 
m+1 n-by-n matrices. 
Phi(m,k)_{i,j} is the number of walks of length m from i->j with exactly k forward edges.
There is a recursive formula to compute them in the paper. 
$\Phi(m,k) = \Phi(m-1,k-1) \mD^{-1}\mA + \Phi(m-1,k) ( \mD^{-1} \mA)^T$
where $\mA$ is the adjacency matrix. 
These get dense quickly, so probably best to store as a dense matrix.

Then we compute approximations using the spectral reconsruction from the Q
lowest frequencies. 

q=(0/(2(m+1)), 1/(2(m+1)), 2/(2(m+1)) ... (Q-1)/(2(m+1)))

Solve for the frequencies using a pseudoinverse on the lienar system from the
paper. (Or check the slides too... it's there in the presentation.json). 

Once we have the approximations
PhiHat(m,k), we compute the different
ApproxErr(m,k) = Phi(m,k)-PhiHat(m,k)
This gives us m+1 n-by-n error matrices.

We first find which value of k gives the highest error

sum_{ij} |ApproxErr(m,k)_{ij}| (i.e. the vector 1-norm of the matrix).

For the plot, we plot

\sum_{ij} |ApproxErr(m,k)_ij|/n (so average over rows...)

Make sense?
